SCROLLABLE DIALOG DEMONSTRATION
═══════════════════════════════════════════════════════════════════

Example Usage:

```python
from ui.widgets.scrollable_dialog import ScrollableDialog

class NewOrderDialog(ScrollableDialog):
    def __init__(self, parent, user_data):
        super().__init__(parent, title="New Order", 
                        width=700, max_height=800)
        
        # Use self.content for your form
        ttk.Label(self.content, text="Customer:").pack()
        ttk.Entry(self.content).pack()
        # ... more fields ...
        
        # Add OK/Cancel buttons
        self.add_ok_cancel_buttons(
            ok_command=self.save,
            cancel_command=self.destroy
        )
```

Benefits:
- Automatically scrolls on small screens
- No content cut-off
- Responsive to screen size
- Consistent behavior
